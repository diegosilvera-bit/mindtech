# MindTech Completo (Estrutura Base)

Sistema em PHP + Bootstrap + MySQL com módulos:
- Usuários
- Clientes
- Equipamentos
- Fornecedores
- Peças
- Estoque
- Ordens de Serviço
- Orçamentos
- Relatórios

## Instalação
1. Extraia em C:\xampp\htdocs\mindtech
2. Importe seu banco de dados mindtech no phpMyAdmin
3. Acesse http://localhost:8080/mindtech/
4. Login: admin / admin
